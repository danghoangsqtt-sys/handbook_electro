## Summary

## Details
**1. Title/Summary**: Cải thiện thuật toán tìm kiếm từ khóa
**2. Which existing feature to enhance?**: Tính năng tra cứu từ điển (Dictionary / TermsList).
**3. Current behavior**: API tìm kiếm dùng exact match `toLowerCase().includes()`, không tìm được tiếng Việt có dấu nếu gõ không dấu. Bỏ sót mảng `applications`. Đọc file đồng bộ từ ổ cứng mỗi lần gọi.
**4. Desired improvement**:
  - Viết hàm chuẩn hóa tiếng Việt loại bỏ dấu (`removeAccents`).
  - So sánh `query` đã bỏ dấu với các fields đã bỏ dấu.
  - Thêm `applications` vào các fields được tìm.
  - Lưu cache nội dung JSON vào memory ở cấp server (`route.ts`) thay vì đọc lại qua `fs`.
**5. Why is this valuable?**: Giúp người dùng Việt Nam tra cứu nhanh chóng, tự nhiên (gõ không dấu). Giảm thiểu đáng kể rủi ro I/O bottleneck khi hệ thống chịu tải.
**6. Breaking changes?**: Không.
