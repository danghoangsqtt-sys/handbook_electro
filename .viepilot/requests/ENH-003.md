# Lỗi / Yêu cầu
1. Thêm tính năng Sort (sắp xếp) trong bảng dữ liệu của trang Linh kiện (`admin/components/page.tsx`) và trang Thuật ngữ (`admin/terms/page.tsx`).
2. Màu sắc màu xanh hiện tại (Neon/Cyan) nhạt và quá chói mắt, cần chuyển sang màu xanh đậm hơn để dễ nhìn.

# Đề xuất giải pháp
1. **Tính năng Sort**:
   - Thêm trạng thái `sortField` và `sortDirection` vào file component/terms admin.
   - Thêm UI click vào Header của table (Tên, Danh mục, Trạng thái...) để toggle hướng sort (Ascending/Descending).
   - Xử lý mảng `users`/`components`/`terms` trước khi map ra UI.
2. **Điều chỉnh màu sắc**:
   - Quét các class sử dụng `text-[#00F0FF]`, `bg-[#00F0FF]/10` (Neon Blue/Cyan) hiện có.
   - Chuyển sang một tone màu xanh dương đậm hơn (VD: `text-blue-500`, `text-[#0066FF]`, hoặc tương tự) để giảm độ chói, đảm bảo độ tương phản trên nền tối.

# Files Affected
- `src/app/(admin)/admin/components/page.tsx`
- `src/app/(admin)/admin/terms/page.tsx`
- `src/app/(admin)/admin/users/page.tsx` (có thể áp dụng sort luôn cho đồng bộ)
