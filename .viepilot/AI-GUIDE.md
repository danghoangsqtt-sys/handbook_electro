# AI Navigation Guide

Welcome, Agent! You are working on the **TechDict AI** project.

## Tech Stack
- **Frontend**: Next.js, React, Tailwind CSS
- **Backend/DB**: Supabase (PostgreSQL)
- **AI Integration**: Gemini API (Google AI Studio)
- **Hosting**: Vercel

## File Relationships
- Architecture Details: `ARCHITECTURE.md`
- Domain Knowledge & Rules: `PROJECT-CONTEXT.md`
- Task Tracking: `ROADMAP.md` and `TRACKER.md`
- Coding Rules: `SYSTEM-RULES.md`
- Database Schema: `schemas/database-schema.sql`

## Core Mechanics
- This app uses a custom "PIN code" authentication instead of standard email/OAuth to keep entry friction near zero.
- The UI is mobile-first, ported from an original monolithic `root.html` prototype.
