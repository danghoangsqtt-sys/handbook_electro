# Brainstorm Session: Sổ tay Bách Khoa Cơ Điện Tử & IT (TechDict AI)
Date: 2026-06-22

## 1. Tầm nhìn & Mục tiêu (Vision)
Xây dựng một web app ứng dụng AI đóng vai trò là "Sổ tay điện tử" toàn diện dành cho kỹ sư, sinh viên và người làm công nghệ.
- Hỗ trợ tra cứu nhanh hơn 3000 thuật ngữ, từ viết tắt đa ngành (Cơ điện tử, Tự động hóa, Khoa học máy tính, Vi xử lý, Điện tử số, IoT, Web, Data, Cyber Security, Hacking, Cryptography...).
- Cung cấp thư viện danh mục các module điện tử phổ thông trong học tập/dự án kèm datasheet, thông số kỹ thuật chi tiết.
- Tích hợp AI (Gemini) đóng vai trò làm Mentor: Hướng dẫn làm đồ án/dự án, đánh giá tính thực tiễn, tính khả thi, lập dự toán chi phí, gợi ý mua linh kiện và cung cấp code mẫu, repo Github tham khảo (kèm step-by-step hướng dẫn).

## 2. Các quyết định thiết kế (Design Decisions)
- **UI/UX Layout:** Kế thừa ý tưởng và cấu trúc từ nguyên mẫu `root.html` hiện tại, nhưng sẽ được đập đi xây lại bằng framework hiện đại, tập trung **tối ưu hóa Responsive chuẩn mực cho Mobile**.
- **Nguồn dữ liệu (Database Seed):** AI sẽ chịu trách nhiệm sinh dữ liệu thuật ngữ tự động (thông qua Script tạo dữ liệu hoặc AI Agent). Yêu cầu nghiêm ngặt về tính chính xác, tính khoa học, chi tiết và liêm chính học thuật.
- **Quản lý người dùng (Authentication):** Đơn giản hóa tối đa. Không yêu cầu đăng ký tài khoản rườm rà (No OAuth/Email), người dùng chỉ cần nhập một **Mã PIN (PIN code)** để truy cập không gian làm việc và lưu trữ lịch sử.
- **Trải nghiệm AI (AI Experience):** Xây dựng một không gian chuyên biệt mang tên **"Phòng Lab AI"** với giao diện khung chat lớn (tương tự ChatGPT/Claude) để người dùng tập trung cao độ vào việc thảo luận chuyên sâu về dự án.
- **Tech Stack:**
  - **Frontend:** Next.js (hoặc Vite + React) + Tailwind CSS (phù hợp tối ưu UI và triển khai Vercel).
  - **Backend/Database:** Supabase (Lưu trữ Dữ liệu thuật ngữ, Modules, Mã PIN người dùng, Lịch sử Chat AI).
  - **AI Engine:** Gemini API (thông qua Google AI Studio API).
  - **Deployment:** Vercel.

## 3. Các giai đoạn phát triển (Phases)

- **Phase 1: Foundation & Mobile-First UI (Tái cấu trúc root.html)**
  - Khởi tạo dự án Next.js, thiết lập Tailwind CSS.
  - Chuyển đổi giao diện từ `root.html` sang kiến trúc Component-based. Tối ưu hóa triệt để trải nghiệm xem trên Điện thoại di động (Mobile).
  - Cấu hình kết nối Supabase, thiết kế Database Schema (Tables: `terms`, `modules`, `users_pin`, `chat_sessions`).

- **Phase 2: Database AI Generation & Core Lookup**
  - Viết NodeJS Script tích hợp Gemini API để sinh tự động kho dữ liệu 3000 thuật ngữ và module điện tử (cài đặt System Prompt đảm bảo tính chuẩn xác khoa học) và nạp (seed) vào Supabase.
  - Xây dựng hệ thống chức năng: Thanh Tìm kiếm (Search), Lọc theo danh mục (Filter), Trang chi tiết thuật ngữ & module.

- **Phase 3: PIN Code Auth & Trải nghiệm "Phòng Lab AI"**
  - Tích hợp logic đăng nhập bằng Mã PIN (kiểm tra từ Supabase) để cá nhân hóa lịch sử.
  - Xây dựng giao diện "Phòng Lab AI" (Chat UI cỡ lớn, hỗ trợ Markdown, Code block).
  - Tích hợp API Gemini vào Phòng Lab AI, xây dựng prompt chuyên biệt hóa giúp AI hiểu bối cảnh Kỹ sư Điện tử/IT, để AI đánh giá dự án, đề xuất module, cung cấp step-by-step guideline.

- **Phase 4: Polish, Testing & Deployment**
  - Review lại toàn bộ tính năng và Mobile UI. Kiểm thử tính ổn định của Phòng Lab AI.
  - Triển khai ứng dụng lên Vercel. Kiểm tra kết nối production với Supabase.
